# Product Overview

Notari is a desktop application that provides cryptographically secure proof-of-work verification through tamper-evident session capture, AI-powered analysis, and blockchain anchoring. 

## Core Purpose
Combat false positives from AI detection tools by providing verifiable evidence of human work through:
- Secure work session capture with tamper-evident timestamps
- AI-powered content analysis and summarization  
- Redactable "Proof Packs" for privacy-controlled sharing
- Blockchain anchoring for immutable verification

## Target Users
- **Students**: Proving original academic work
- **Professionals**: Demonstrating authentic work output
- **Creators**: Validating original content creation
- **Verifiers**: Teachers, employers, clients needing to validate work authenticity

## Key Value Propositions
- Tamper-proof evidence of work processes
- Privacy controls through selective redaction
- Independent third-party verification
- Cross-platform desktop support (Windows/macOS)
- Flexible pricing with subscription and pay-per-use options
